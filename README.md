## 깃허브
- main은 사용 금지!
- Default 브랜치는 devleop!
- 이슈 생성 후 주어지는 #번호 를 보고 번호에 따른 새 브랜치 생성
- 작업
- 해당 번호 브랜치에서 작업 완료 후 커밋하고 폴 리퀘스트 생성
- 해당 작업한 제외 다른 인원이 코드 리뷰 후 approve
- 3명이 채워지면 머지하기! (develop브랜치로 머지됨, 해당 번호 브랜치는 자동 삭제)
